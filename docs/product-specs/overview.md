# プロダクト概要

## ミッション

日本人旅行者が「このパスポートでここに行けるか」を、
5秒以内に正確に把握できる世界を作る。

---

## プロダクト名（仮）

**VisaCheck** — シンプルに、すぐわかる。

---

## ターゲットユーザー

| セグメント | 説明 |
|---|---|
| 個人旅行者 | バックパッカー・旅行好き・ノマドワーカー |
| 直前手配層 | 旅行直前にビザ要否を確認したい人 |
| 旅行代理店 | 顧客への案内に使いたい業者（将来の法人プラン） |

**初期フォーカス：日本国籍パスポートを持つ個人旅行者**

---

## 解決する課題

### 現状の痛点
- 外務省サイトは情報が探しにくく、UIが古い
- 既存の比較サイトは情報が古くて信頼できない
- 「ビザなしで行けるか」をすぐ知りたいのに、調べるのに時間がかかる
- 入国条件（残存有効期間・往復航空券など）が一覧できない

### VisaCheckが提供する価値
- 国名を入れるだけで**即座**にビザ要否がわかる
- 情報は**自動更新**で常に最新
- ビザ取得方法・費用・注意事項まで**ワンストップ**で確認できる

---

## 主要機能一覧

### MVP（最初にリリースするもの）

| 機能 | 説明 | 優先度 |
|---|---|---|
| ビザ要否検索 | 渡航先を入力→ビザ要否を即表示 | 🔴 必須 |
| ビザ情報詳細 | 取得方法・費用・滞在可能日数・注意事項 | 🔴 必須 |
| 国一覧ページ | 全渡航先の一覧・フィルター | 🟡 重要 |
| 情報の更新日表示 | 最終更新日時を明示して信頼性を担保 | 🟡 重要 |

### Phase 2（MVP後）

| 機能 | 説明 |
|---|---|
| お気に入り登録 | よく調べる国をブックマーク |
| 旅程チェック | 複数国の旅程を一括チェック |
| 通知機能 | 入国条件変更をメール通知 |
| 多言語対応 | 英語・中国語・韓国語 |
| 法人プラン | 旅行代理店向けAPI提供 |
