# UXフロー

## 基本的なユーザージャーニー

```
トップページ
  └─ 国名を入力（サジェスト付き）
        └─ 検索ボタン押下
              └─ 結果ページ
                    ├─ ビザ不要 → 滞在可能日数・注意事項を表示
                    ├─ ビザ必要 → 取得方法・費用・申請先を表示
                    └─ 要確認  → 外務省リンクへ誘導
```

---

## 画面一覧

### 1. トップページ（`/`）

**目的**：渡航先を入力して検索させる

**構成要素**
- ロゴ・キャッチコピー
- 国名検索フォーム（インクリメンタルサーチ）
- よく検索される国（人気順ショートカット）
- フッター（更新日・免責事項）

**UXのポイント**
- 検索フォームが画面中央に来るようにする
- 国名は日本語・英語どちらでも検索できる
- モバイルファーストで設計する

---

### 2. 検索結果ページ（`/result/[countryCode]`）

**目的**：ビザ情報をわかりやすく伝える

**構成要素**

```
[国名] への渡航情報

━━━━━━━━━━━━━━━━━━━━━━━━
✅ ビザ不要（または ⚠️ ビザ必要 / ❓ 要確認）
━━━━━━━━━━━━━━━━━━━━━━━━

📋 基本情報
  - 滞在可能日数：90日
  - パスポート残存有効期間：6ヶ月以上
  - 往復航空券：必要

💰 費用
  - ビザ申請料：無料（ビザ不要の場合は不要）

📝 取得方法
  - 申請方法・必要書類・申請先

⚠️ 注意事項
  - 最新の入国条件の変更点など

🔗 公式情報
  - 外務省・現地大使館リンク

最終更新：2026年3月22日
```

**UXのポイント**
- ビザ要否は**一番上・大きく・色で**伝える（緑/赤/黄）
- 詳細情報はアコーディオンで折りたためる
- 「この情報をシェア」ボタンを設置

---

### 3. 国一覧ページ（`/countries`）

**目的**：全渡航先を一覧・フィルターできる

**構成要素**
- 地域フィルター（アジア・ヨーロッパ・アフリカ等）
- ビザ要否フィルター（不要のみ・必要のみ）
- 国カード一覧（国旗・国名・ビザ要否バッジ）

---

## エラー状態

| 状態 | 表示内容 |
|---|---|
| 該当国なし | 「見つかりませんでした。国名を確認してください」 |
| データ未取得 | 「現在情報を取得できません。外務省サイトをご確認ください」|
| 情報古い可能性 | 「この情報は30日以上更新されていません。公式サイトも確認してください」|

---

## モバイル対応方針

- ブレークポイント：`sm: 640px` / `md: 768px` / `lg: 1024px`
- タップターゲット最小サイズ：44px × 44px
- 検索フォームはモバイルでも使いやすいよう、キーボードが出ても見切れない設計にする
